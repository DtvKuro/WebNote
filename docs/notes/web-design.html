COLOR THEORY

Mood of color - (IMPORTANT) Tells a story or conveys message.

Examples:
Red - Intensity or Energetic
Yellow - Joy or Attention Grabbing <Bad for long term usage, too attention grabbing>
Green - Fresh or Safety
Blue - Stability or Trust
Purple - Royalty, Wealth and Femininity

Combining Colors

ANALOGOUS COLORS - Colors that are next to each other on the color wheel. (DOESNT STAND OUT)
Good for navigation bars, logo and backgrounds.
COMPLEMENTARY COLORS - Colors that are opposite each other on the color wheel. (STANDS OUT)
Dont use for text and text background.

ADOBE COLOR WHEEL - https://color.adobe.com/create/color-wheel
You can create your own color palette using the color wheel.

COLOR HUNT - https://colorhunt.co/
You can find color palettes that are already made by other people.

TYPOGRAPHY

Serif has feet - (Times New Roman)
Serious, Old, Authoritative, Traditional, Stable, Respectable

Sans Serif - (Arial)
Sensible, Simple, Straightforward. Soft, Friendly, Readable

Stick to 2 FONTS FOR CLEANER READABILITY
AND RELATE THE FONTS TO THE MOOD & TIME ERA.

AND CONTRAST TO SERIFNESS. Heading is serif and sans serif to the body. USE WEIGHT TO CONTRAST THE FONTS.

USER INTERFACE (UI) DESIGN

Manage ATTENTION with effective User Interface (UI) Design.

  1. Hierarchy - can be conveyed through
    Color, Size,
  2. Layout - Create different positions to make it interesting
    Text Length
  3. Alignment - Aligning to make everything look connected.
    In order to look professional, reduce the alignment point.
  4. White Space - Indicate Importance or Luxury.
  5. Audience - You will design based on your target audience.

USER EXPERIENCE (UX) DESIGN

  1. Simplicity - Ease the user by not making user feel overwhelm.
  2. Consistency - Function and Design consistent
  3. Reading Pattern - People usually reads in F pattern or Z Pattern(Faster viewing). Identify how user reads your website.
  4. All Platform Design - Design for Mobile too. Mobile Responsive.
  5. Dont use your Powers for evil - Tricks the user to do something they dont want to.
